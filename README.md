# PROND
Code for Reconstructing Diffusion Network Topologies from Partial Observations
=======================================================================================================================================
PROND: Main implementation of the paper "Reconstructing Diffusion Network Topologies from Partial Observations"
Simulate Infection Data: Code for simulating infection data under the Independent Cascades (IC) model
benchmark: Code for generating synthetic LFR benchmark network
generate_memetracker_dataset: Code for processing MemeTracker dataset to obtain the real-world network and diffusion results/infection data
=======================================================================================================================================
